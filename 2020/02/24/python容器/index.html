python各种容器

应用

删除列表中的变量：del(a[x])或删除切片形式:a[a:b]
列表：列表是（可变）序列，是容器。
相关函数：len,max,min,sum,any,all
相关方法：index,insert,count,remove,copy(deepcopy),append,extend(list),clear,sort,pop(索引)，reverse(反转)
字符串方法：.split(),.join.
列表推导式：【x for x in range(10)】,通过推导式生成一个从推到始终得到的数据的列表。可以嵌套多重循环

元组 tuple

元组是不可变的序列，但是是可以存放任意类型数据的容器。
所有的序列都可以迭代，（字符串，列表，元组），所有的容器也都是可以迭代的。
元组不能改变，所以不支持索引赋值和切片赋值，但是支持‘+’号拼接，拼接后会产生一个新的元组替代原元组。
元组排序:sorted（iterable，key=None，reverse=Ture）生成一个可迭代对象，
reversed（可迭代对象），返回反向顺序的可迭代对象，倒置。生成一个可迭代对象，仅可用于迭代或推导式。

容器小节：

字符串str #不可变序列，只能存字符
列表 list #可变，可以存任何数据
元组 tuple #不可变 可以存任何数据

字典

字典的键必须是不可变类型：bool,int,float,complex,str,tuple,frozenset,bytes
字典的值可以为任何类型。

可变类型有四种：list，dict，set，bytearray（字节数组）

集合 set：

可变容器
集合内数据对象唯一，不可重复
无序存储结构，数据没有先后关系
集合内元素必须是不可变对象
相当于没有键值的字典。
