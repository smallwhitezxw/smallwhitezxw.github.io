python函数

函数

  • 用于封装语句块，提高代码的复用性

  • 定义用户级别的函数

语法

  • 基本说明：

    1. 函数名字就是语句块的名称
    2. 命名规则与变量名相同
    3. 函数名是一个变量（不要轻易对其赋值）
    4. 函数有自己的名字空间，在函数外部不可以访问函数内部的变量，在函数可以访问函数外部的变量
    5. 语句部分不能为空，如果为空需要填充pass

  • 函数调用：

    1. 函数调用是一个表达式
    2. 如果函数内部没有return语句，函数调用完毕返回None

  • 函数传参(函数调用所传的实参)：

    1. 序列传参：在可遍历序列前加*，

    2. 关键字传参：通过形参的名字匹配实参。

    3. 字典关键字传参：将字典用‘**{}’，拆解后在进行关键字传参的传参方式。

      • 说明：

        1. 字典的键名和形参名必须保持一致

          2. 字典的键名必须是字符串
          1. 字典的键名要在形参中存在

    4. 位置传参：最常用最基本的传参方式。

函数形参的定义：

  • 缺省函数说明：缺省参数必须自右至左依次存在，如果一个参数有缺省参数，则右侧的所有参数必须有缺省参数。

  • 函数形参的定义方式：

    1. 位置形参:def 函数名（形参名1，形参名2，…）

    2. 星号元组形参：def 函数名（*元组形参名）

      • 作用：收集多余的位置传参

    3. 关键字传参：def 函数名（，命名关键字传参）或def 函数名（\元组形参名，命名关键字传参）

      • 作用：必须使用关键字传参或字典关键字传参。

    4. 双星号字典形参：def 函数名（**字典形参名）：

      • 作用：收集多余的关键字传参
      • 字典形参名通常命名为’kwargs’

  • 函数的参数说明：

    1. 四种形参方式可以混合使用，函数形参自左到右的顺序为：1位置形参，2星号元组形参，3命名关键字形参，4双星好字典形参。
