mysql环境再配置（压缩包）

配个鸡儿的mysql搞了一下午就离谱！记下来，防止下次GG。

由于之前在官网下载的mysql由于实在是卸载不干净了，根本没法用。所以用的压缩包安装。

  1. 下载完压缩包后解压到想要安装的路径下。G:\mysql\mysql-8.0.18-winx64。

  2. 新建一个名为my.ini的配置文件。覆盖到解压的目录下(G:\mysql\mysql-8.0.18-winx64)。

    • my.ini配置如下：

  3. 以管理员方式打开powerShell，进入到安装的bin目录下(G:\mysql\mysql-8.0.18-winx64\bin)，执行mysqld –initialize –console(.\mysqld –initialize –console)初始化命令，执行完成后会打印root用户的初始默认密码。保存到记事本中。

    • 可能会产生错误，由于my.ini中的basedir和datadir配置错误导致不能成功初始化，我当时配置成了G:\mysql。！！！(血的教训！)

  4. 然后在powerShell在执行mysqld -install(.\mysqld -install),

  5. 配置环境变量，将bin目录添加到path下(G:\mysql\mysql-8.0.18-winx64\bin)。

  6. 在powerShell中执行net start mysql开启mysql服务。

  7. 登录mysql：

    1. mysql -u root -p
    2. 这是会输入密码，用之前记载记事本中的密码登录即可。
    3. 登陆成功后，通过ALTER USER ‘root’@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘新密码’; 语句修改密码。

  8. 完成！！！！！！！！！！！！！！！！！！！！！！！！！
